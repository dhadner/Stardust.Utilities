How to fix this in Microsoft Edge

Edge's Enhanced Security Mode can disable WebAssembly. To allow it for this site:

  1. Navigate to edge://settings/privacy/security/secureModeSites
  2. Add this site to the "Never use enhanced security for these sites" list
  3. Come back and click Try again above

Chrome and Firefox are not affected.
